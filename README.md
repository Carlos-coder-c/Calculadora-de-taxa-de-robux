Calculadora de Taxas de Robux (Criadores Roblox)

Uma ferramenta simples e objetiva desenvolvida para criadores do Roblox calcularem rapidamente o valor líquido recebido após as taxas da plataforma, ou o valor bruto necessário para receber um valor desejado.

O Roblox cobra 30% de taxa sobre vendas de itens como roupas, game passes e open-view. Este utilitário elimina a necessidade de cálculos manuais e reduz erros.

Funcionalidades

Seleção de categoria com taxa aplicada automaticamente.

Cálculo do valor líquido recebido a partir do preço bruto.

Cálculo do preço bruto necessário para atingir um valor líquido desejado.

Interface em terminal, simples e direta.

Categorias Disponíveis

roupas

game-pass

open-view

Todas possuem taxa padrão de 30%.

Como Funciona
1. Calcular valor líquido

Informe o preço bruto do item.
O programa retorna quanto você realmente recebe após a taxa.

2. Calcular valor bruto

Informe quanto deseja receber.
O programa calcula qual deve ser o preço bruto do item para atingir esse valor.

Observações Importantes

Informe somente o valor em Robux, não o preço em dinheiro.

A taxa utilizada é sempre 30%, padrão do Roblox.

Ideal para criadores que desejam precificar itens com precisão.

Autor

Projeto criado para ajudar a comunidade de criadores Roblox que precisam de cálculos rápidos e confiáveis sobre as taxas da plataforma.
